Word Search Solver

Android and Windows App to solve word searches (find-a-words) from an image.
Created with Python using Kivy, Open CV

Reading the puzzle

The vision system takes an image and gives a grid of letter posibilities,

  • First it preprocesses the image by applying a variety of effects including, blurring, greyscaling, thresholding and scaling down
  • Second it finds all contours and filters them down to ones that are roughly the right size and shape
  • Third it creates a small images fitted to each of the contours and places them into a grid
  • Fourth it passes each image into a K Nearest Neighbour Classifer. The Knn classifier works by finding
    the summed pixel difference between the given image and every image in an autogenerated dataset of labeled letters and gives you a list of the letters that match up the best.

Solving the puzzle

  • Finding all Knn matches for the first letter
  • For each first letter found check all adjacent grid squares to see if they had any matches with the second letter
  • Continue checking for all remaining letters in the same direction
  • If you find all letters in a word, assign it a confidence based on how many of the Knn matches were for the letters used.
  • The the word with the highest confidence.

App

Made with the python library Kivy, builds to andorid with buildozer (part of kivy) and pyfreeze for windows.
