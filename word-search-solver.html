Word Search Solver

Android and Windows App to solve word searches (find-a-words) from a picture.
Created with Python using Kivy, Open CV

Vision

The Final Vision system works by:

  • First asking the user to select the area which the actual grid is, this was done instead of trying
    to automatically determin where it is to increase reliability but does mean the system can not work on a live feed
  • Second it preprocesses the image by applying a variety of effects including, blurring, greyscaling, thresholding and scaling down
  • Third it finds all contours and filters the ones of roughly the right size
  • Fourth It creates a small images fitted to each of the contours and passes them into a K Nearest Neighbour Classifer. The Knn classifier works by simply finding
    the summed pixel difference between the given image and every image in an autogenerated dataset of labeled letters and gives you which letters it matches up to the best.
  • Fifth it generates a grid by allocating letters based on their position in the image
  • Finally it finds the given words in the grid by just checking for every grid location if any of the words are in any direction and annotating the origonal image to display back
